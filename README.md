## ⚙️ Configuration

Édite le fichier `config.json` :

- `token` : ton token Discord
- `prefix` : le préfixe des commandes (par défaut `-`)
- `logsChannel` : ID du salon où les logs seront envoyés
- `raidLimit` : nombre de joins rapides avant détection de raid
- `spamLimit` : nombre de messages rapides avant mute
- `mentionLimit` : nombre de mentions avant mute

---

## 🔧 Commandes disponibles

### Modération
- `-warn @user raison`
- `-mute @user minutes raison`
- `-unmute @user`
- `-kick @user raison`
- `-ban @user raison`
